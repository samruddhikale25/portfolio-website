# Personal Portfolio Website

## Project Title
Personal Portfolio Website

## Created By
Samruddhi Kale

## Objective
This project is a personal portfolio website developed to showcase my skills, education, projects, and contact details in a professional and responsive way.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- GitHub

## Features
- Fully responsive design (Mobile / Tablet / Desktop)
- Navbar with toggle menu for mobile view
- Sections: Home, About, Education, Skills, Internship, Certifications, Projects, Contact
- Project modal popup with details
- Contact form with validation
- Modern UI design with hover effects

## Project Structure
- index.html
- style.css
- script.js
- README.md

## Challenges Faced
- Implementing modal popup using JavaScript
